# New Ankle Dubai V3

Production-ready multilingual static Next.js website for Cloudflare Pages.

## Local build

```bash
pnpm install
pnpm run build
```

## Cloudflare Pages

- Build command: `pnpm run build`
- Build output directory: `out`
- Root directory: leave empty
- Node.js: 22 or later

## Included

- English, Russian and Arabic routes
- Premium homepage
- Medical Director profile for Armen Hagopjanian, DPM, FACFAS
- 6 Centers of Excellence
- 27 condition pages in three languages
- 12 procedure pages in three languages
- International patients pathway
- Consultation and contact forms (mailto draft until a secure form backend is connected)
- SEO sitemap and robots.txt
- Responsive mobile layout
