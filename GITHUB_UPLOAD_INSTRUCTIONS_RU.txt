КАК ЗАГРУЗИТЬ ПРОЕКТ ЧЕРЕЗ GITHUB DESKTOP

1. Распакуйте архив в отдельную папку, например:
   C:\NewAnkleDubai

2. В GitHub Desktop выберите репозиторий:
   LUXURYAI/newankledubai

3. Сначала клонируйте пустой репозиторий на компьютер.

4. Откройте папку клонированного репозитория.

5. Скопируйте ВНУТРЬ неё все файлы из папки этого проекта.
   Копировать нужно содержимое папки, а не саму внешнюю папку.

6. Вернитесь в GitHub Desktop.
   Программа покажет список добавленных файлов.

7. В поле Summary напишите:
   Initial New Ankle Dubai website

8. Нажмите:
   Commit to main

9. Затем нажмите:
   Push origin

После этого проект появится в GitHub и будет готов к подключению к Cloudflare Pages.
